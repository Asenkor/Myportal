HyperText Markup Language, язык гипертекстовой разметки.

HTML - это основа основ. Это не программирование, это всего лишь основа верстки, чуть усложненная версия кнопочек в MS Word. С помощью этого языка мы создаем скелет страницы, его структуру. Простыми словами, мы указываем браузеру, что на странице есть заголовок, таблица, изображение, список и тому подобное, но не указываем где они находятся, как оформлены и расположены. Набросок карандашом, если хотите.

Для изучения используем следующие источники:

  • htmlbook.ru - самоучитель по html 4 и введение в html5
  • htmlacademy.ru - интерактивные курсы
  • webref.ru - самоучитель по html
  • потренироватся
  • опционально: книга "Погружнние в html5" Марка Пилигрима

Что я должен знать в итоге?

  • семантика
  • структура
  • форматирование
  • микроразметка
  • другие разделы html5, особенно касающиеся блоков, слоев и т.д.

CSS или каскадные таблицы стилей.

Выделены для облегчения написания HTML-разметки и разделения "разметки-оформления". Предназначены для визуального оформления страницы. Если в HTML мы наросали разметку сайта карандашом, то CSS позволяет нам разукрасить наш рисунок и расположить композицию наиболее удачно в конкретном случае. Такая же основа, как и HTML.

Изучать желательно в следующем порядке.

  • htmlbook.ru - самоучитель CSS, введение в CSS3
  • htmlacademy.ru - основы CSS
  • webref.ru - основы CSS, блочная модель в CSS, текст в CSS позиционирование в CSS и продвинутый CSS
  • изучить препроцессоры например scss
  • uroki-css.ru - введение в анимацию в CSS
  • уроки на англ языке - dev tips (канал на ютубе)
  • книга "Большая книга css3" Дэвид Макфарланд
  • книга "Css. Каскадные таблицы стилей. Подробное руководство" Эрик Мейер

Javascript, motherfucker!

В конце концов, его величество javascript. С его помощью можно делать на сайте все, что необходимо. По сути весь функционал сайта состоит из написанных на js скриптов. Учить до этого уровня обязательно.

План обучения js таков:

  • стандарт ecma script es5
  • сайт: learn.javascript.ru
  • книга: Дэвид Флэгман- JavaScript. Подробное руководство
  • необходимо изучить библиотеку jQuerry
