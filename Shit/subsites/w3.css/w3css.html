Привет, я сайт на W3.CSS!

Меня сделали для того, чтобы научиться верстать с помощью фрэймворка

W3.CSS - это фреймворк. Любой фреймворк, будь то Bootstrap, Foundation и прочие, создан для того, чтобы облегчить верстальщику задачу. Это инструмент, который позволяет думать о том "что реализовывать", а не о том "как реализовывать". Это набор кода, который вшит в файл стилей, специально разработанный для использования другими разработчиками, со своим собственным функционалом, такая веб-версия инкапсуляции из программирования. Верстальщику доступны только названия классов и описание того, что этот класс делает. Все остальное скрыто от глаз.
